<h2><a href="https://leetcode.com/problems/find-the-duplicate-number/description/">287. Find the Duplicate Number</a></h2><h3>Medium</h3><hr><div><p>Given an array of integers nums containing n + 1 integers where each integer is in the range [1, n] inclusive.

There is only one repeated number in nums, return this repeated number.

You must solve the problem without modifying the array nums and uses only constant extra space.

<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> nums = [1,3,4,2,2]
<strong>Output:</strong> 2

<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> nums = [3,1,3,4,2]
<strong>Output:</strong> 3

<p><strong>Example 3:</strong></p>
<pre><strong>Input:</strong> nums = [3,3,3,3,3]
<strong>Output:</strong> 3

<p>&nbsp;</p>
<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 <= n <= 10^5</code>.</li>
	<li><code>nums.length == n + 1</code></li>
    <li>1 <= nums[i] <= n</li>
    <li>All the integers in nums appear only once except for precisely one integer which appears two or more times.</li>
</ul>
</div>
